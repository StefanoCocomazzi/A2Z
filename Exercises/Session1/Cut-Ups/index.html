The Cut-Up Machine

By Stefano Cocomazzi

This Machine will cut a text in different pieces and then rearranges them in order to create a poem.
The method was invented by William S. Burroughs. Read the article on Wikipedia

Instructions:

  1. Type or Copy some text below
  2. Select Poem Length
  3. Press Cut-Up
  4. Congratulations! Now you are a poet
