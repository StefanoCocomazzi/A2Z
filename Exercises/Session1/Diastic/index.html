Jackson Mac Low Diastic Machine

By Stefano Cocomazzi

This machine will generate a poetry from a text and a seed. For each world in the seed, this will generate a row in the poetry. It's hard to explain, just enjoy it.

  1. Type or Copy the source code
  2. Type the Seed
